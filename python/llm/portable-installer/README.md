# BigDL-LLM One-Click Installer For Windows

## Supported models

- ChatGLM2-6b
- Baichuan-13B-Chat
- Baichuan2-7B-Chat

## How to use

1. Run `setup.bat` and wait until it finished.
2. Run `chat.bat` and input the path of the model (e.g. `path\to\model`, note that there's no slash at the end of the path).
